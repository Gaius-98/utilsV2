# utilsV2
gaius发布的工具
# 安装
```
npm i gaius-utils-v2
```
# 使用
1. 全部引入
  ```
  import GaiusUtilsV2 from 'gaius-utils-v2'
  import 'gaius-utils-v2/lib/style.css'
  Vue.use(GaiusUtilsV2)
  ```
